# CaptLive

基于 StreamCap 和 DouyinLiveWebFetcher 合并开发的直播录制工具。

## 📋 项目说明

本项目由两个开源项目合并并修改而来：

1. **StreamCap** - 多平台直播录制工具
2. **DouyinLiveWebFetcher** - 抖音网页版弹幕数据抓取

详细代码来源和作者信息请查看 [ATTRIBUTION.md](./ATTRIBUTION.md)

## 🚀 快速开始

### 环境要求

- Python 3.10+
- FFmpeg
- Node.js v18+

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行

```bash
python main.py
```
